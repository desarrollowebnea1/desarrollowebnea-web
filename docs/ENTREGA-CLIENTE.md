## Instrucciones básicas

### 1. Ingresar al admin

1. Ir a `/admin/login`
2. Ingresar email y contraseña
3. Acceder al dashboard

### 2. Agregar un proyecto al portfolio

1. Admin → **Proyectos** → **Nuevo proyecto**
2. Completar: nombre, rubro, descripción, link web, tecnologías
3. Subir imagen desde **Imágenes** o pegar URL
4. Activar y marcar como destacado si corresponde
5. Guardar — aparece automáticamente en la web pública

### 3. Editar servicios y precios

1. Admin → **Servicios** — editar nombre, descripción, precio desde
2. Admin → **Planes y precios** — editar planes, features y precios
3. Los cambios se reflejan en la web sin tocar código

### 4. Revisar solicitudes de presupuesto

1. Admin → **Solicitudes**
2. Ver detalle de cada solicitud (código DWN)
3. Cambiar estado: Nueva → Contactada → En análisis → Presupuestada → Cerrada
4. Responder por WhatsApp desde el botón integrado

### 5. Subir imágenes

1. Admin → **Imágenes**
2. Subir JPG, PNG o WEBP (máx. 5 MB)
3. Copiar URL y usarla en proyectos, configuración o testimonios

### 6. Editar textos principales

1. Admin → **Configuración**
2. Editar hero, slogan, contacto, SEO, textos de soporte
3. Guardar cambios

### 7. Gestionar FAQ y testimonios

- **FAQ:** Admin → FAQ — agregar, editar, desactivar preguntas
- **Testimonios:** Admin → Testimonios — agregar reseñas de clientes

## Consulta de estado (cliente final)

Los clientes pueden consultar su solicitud en la web pública:

1. Sección **Presupuesto** → **Consultar estado**
2. Ingresar código `DWN-YYYYMMDD-XXXX`
3. Ver estado actual y mensaje
